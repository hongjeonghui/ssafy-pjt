## 최종 프로젝트 구현
- **추천**이라는 대주제 내에서 구현할 것
- 필수 기능 및 도전 과제 기능을 각자 확보한 데이터 셋에 적용하여 구현해야 함

## 제출 시 유의사항
- 준비한 데이터 셋 함께 제출할 것
  - 합성 데이터 활용 가능하나, 제한적으로 사용할 것
- 개발 환경
  - Django, Vue만 가능
  - Spring, React 등은 사용 불가하되, 그 외 API, 라이브러리, 패키지 등은 자유롭게 활용 가능

## 주제 변경 조건
- 서비스를 위한 데이터 셋을 확보하지 못한 경우만 가능
- 기한 : 12월 12일
  - 당일 신청만 가능 (사전 변경 불가)
- 자율 프로젝트의 경우 주제 변경되는 것은 상관 없음
